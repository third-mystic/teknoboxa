# Teknoboxa - Neon Arenas

## Project Description
Teknoboxa - Neon Arenas is a cyberpunk pocket-sized tabletop fighting game featuring brutal simultaneous combat with hand management and tactical grid movement.

**Game Stats:**
*   **Players:** 2
*   **Ages:** 14+
*   **Playtime:** 45 Minutes

Are you ready to step into the Arena?

## Website Structure
*   `index.html`: The main landing page for Teknoboxa - Neon Arenas.
*   `css/styles.css`: Contains the styling for the website, including a cyberpunk aesthetic.
*   `fonts/`: Custom fonts used in the design.
*   `img/`: Image assets for the website.
*   `js/script.js`: JavaScript for interactive elements, such as the email subscription form.
